# Amazon Sales Data Analysis
## 📋 Project Overview
Is project mein Amazon ke products ki sales aur discount analysis ki gayi hai using MySQL, Pandas, NumPy, aur Power BI.
## 🛠️ Tech Stack
- Power BI (Dashboard)
- MySQL (Database)
- Python (Pandas, NumPy)
## 📈 Key Insights
- Top selling categories
- Discount vs Sales trends
## 🚀 How to use
1. Download the .pbix file.
2. Open in Power BI Desktop.
3. Data refresh karein.
